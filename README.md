# Work Day Scheduler 

## Description

 __This project was about creating a simple calendar application that utilizes local storage in order to save events according to each hour of the working day.__  

* Languages utilized:
  * HTML
  * CSS
  * JavaScript
  * jQuery
  * Bootstrap
  * MomentJS
